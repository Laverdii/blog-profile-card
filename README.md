# Profile Card Project

A responsive profile card built using HTML and CSS, focused on layout structure, spacing consistency, and visual hierarchy.

## 🚀 About the Project

This project was developed to strengthen my understanding of core web fundamentals, including box model behavior, spacing control, flexbox alignment, and visual balance.

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Git & GitHub

## 🎯 Key Concepts Practiced

- Box model and spacing control (margin vs padding)
- Shadow balance and visual proportion
- Text hierarchy and secondary typography
- Basic version control workflow using Git

## 📌 What I Learned

Building this project helped me understand how layout decisions affect visual balance, and how version control is essential for structured development.

Made with focus on fundamentals and clean structure.
